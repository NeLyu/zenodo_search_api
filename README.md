# zenodo_search_api
That's API for AI powered search in Zenodo database.
Built with FastAPI.

## Installation

Clone the repository and install the dependencies:

```
git clone https://github.com/NeLyu/channel_fetch.git
cd channel_fetch
pip install -r requirements.txt
```
